IP Location

É possível que você já tenha se deparado com a sigla IP em algum momento, navegando pela internet, configurando seu smartphone ou até mesmo passeando pelas funcionalidades de aplicativos. Internet Protocol, ou endereço de IP, é o é um número de identificação de aparelhos que se conectam à internet, como é o caso de computadores, smartphones e dos roteadores sem fio, que fazem a distribuição do sinal de WiFi na sua casa ou escritório.

O IP Location é utilizado para descobrir a localização e outras informações de um endereço de IP externo. IP externo é a identificação gerada a partir da conexão do seu dispositivo com uma rede. Assim como o CPF, cada equipamento gera um protocolo de identificação diferente quando conectado a internet, logo, não existe duplicidade de IPs entre redes e dispositivos.

(Deixe vazio para usar o próprio IP).
